# 麻雀ルール共有アプリ：フェーズ1＆フェーズ2 総合設計書

本ドキュメントは、麻雀ルール共有アプリのデータ定義（デフォルトルール・比較用基準ルール）およびUI/UXの画面・出力レイアウト仕様をまとめたものである。

## 【フェーズ1：タスク1】 デフォルトルール（四人麻雀 PDF準拠版）

* **1. 基本設定・ポイント精算**
* 対局の長さ: 東南戦 
* 持ち点 / 返し点: 25,000点持ち / 30,000点返し 
* ウマ: 10-20 
* トビ終了: あり（0点は続行） 
* 西入: 無し 
* 一本場: 300点 

* **2. 進行・連荘ルール**
* 連荘条件: テンパイ連荘 
* オーラスのアガリやめ・テンパイやめ: OK 
* 途中流局: 有り（九種九牌、四家立直、四槓散了、四風連打、三家和が対象） 
* 途中流局時の本場: 本場積む 
* ダブロン・トリロン: あり（本場、積み棒、連荘権は頭ハネ） 
* 鳴きの優先順位: 発声優先（同時の場合はポン優先） 

* **3. 役・ドラ**
* 喰いタン・後付け: 喰いタンあり、後付けあり（アリアリ） 
* 喰い替え: 禁止 
* 赤ドラ: 各1枚 
* 切り上げ満貫: あり 
* 連風牌の対子: 2符加算 
* 流し満貫: あり（流局時成立） 
* 国士無双の暗槓上がり: あり 
* 役満の種類: 雀魂に準ずる / 役満の複合・W役満: 無し 
* 数え役満: 13飜以上 

* **4. リーチ関連**
* フリテンリーチ: あり 
* オープンリーチ: 無し 
* ツモ番の無いリーチ: 不可 
* 1000点未満でのリーチ: 不可 
* リーチ後の暗槓: 待ちが変わらなければ可 

* **5. 祝儀（チップ）設定**
* 対象役: 一発、裏ドラ、赤ドラ（鳴き祝儀あり） 
* 役満祝儀: ツモは5枚オール、ロンは10枚 
* 祝儀なしの対象: 数え役満、オールスター、焼き鳥、トビ賞 

* **6. パオ（責任払い）**
* 適用条件: 役満を確定させる牌を鳴かせた場合 
* 支払い割合（ツモ）: パオの人の全額払い 
* 支払い割合（ロン）: 他の人が振り込んだ場合、パオの人と折半 

* **7. 罰則・ペナルティ設定**
* チョンボ: 誤ロン・誤ツモで倒牌、ノーテンリーチで流局、リーチ後に待ちの変わる暗槓 
* 上がり放棄: 誤ロン・誤ツモで倒牌なし、誤ポン・誤チー・誤槓で牌を晒した場合、鳴きでの晒し間違い、多牌 
* 軽罰符（場に1000点供託）: 誤ポン・チー・槓（発生のみ）、リーチ取消（条件あり）、少牌 

## 【フェーズ1：タスク1】 デフォルトルール（三人麻雀 PDF準拠版）

* **1. 基本設定・ポイント精算**
* 対局の長さ: 東南戦 
* 持ち点 / 返し点: 35,000点持ち / 40,000点返し 
* ウマ: 1位+20 / 2位±0 / 3位-20 
* トビ終了: あり（0点ちょうどもトビ扱い、ゲーム終了） 
* 得点計算: ツモ損なし、符計算なし（専用点数表を使用）、ツモピンあり 
* 一本場: 1,000点（上家取り） 
* 流局時テンパイ料: 場に2,000点 

* **2. 進行・連荘ルール**
* 連荘条件: テンパイ連荘（形式テンパイOK） 
* オーラスのアガリやめ・テンパイやめ: あり（トップだと強制終了） 
* 途中流局: なし（3人リーチも続行） 
* ダブロン: あり 
* ツモ山: ドラ表示牌の隣まではツモり切り 

* **3. 役・ドラ・牌の扱い**
* 喰いタン・後付け: あり 
* 赤ドラ・虹ドラ: 5索・5筒それぞれに赤2枚＋虹2枚（計8枚、全てドラ扱い） 
* 北の扱い: 共通役牌（北抜きなし） 
* 花牌: あり 
* カンドラ: 先めくり 

* **4. リーチ関連**
* 1,000点でのリーチ: 1,000点でも可能（流局でトビ） 
* フリテンリーチ・見逃しツモ: 可能 
* オープンリーチ: あり（全晒し、1,000点で可能、フリテン可） 
* リーチ後のカン: 待ちが変わらなければOK 

* **5. 特殊役・ローカルルール**
* 2軒リーチ時の手牌公開: 2軒リーチが入った場合、残り1人は手牌を公開する 
* 白ポッチルール: リーチ後オールマイティ扱い 
* 特殊役満: マンズ混一色、大車輪、人和、流し役満 
* 特殊役: 四枚使い七対子（2翻）、小車輪（混一七対子・6翻） 
* 役満の扱い: 数え役満は14翻から W役満はなし 

* **6. 祝儀（チップ）設定・パオ**
* 祝儀対象: 一発、裏ドラ、虹牌（鳴き祝儀）、リーチ後白ポッチツモ 
* 役満祝儀: ツモはチップ5枚、ロンはチップ10枚 
* トビ賞: 飛ばされた人が場にチップ2枚を出し分配 
* パオ: 大三元、大四喜に適用 

* **7. 罰則・ペナルティ設定**
* チョンボ: 誤ロン・ツモで倒牌、ノーテンリーチ流局、リーチ後の待ち変更暗槓 
* 上がり放棄: 誤ロン・ツモ発声のみ、誤ポン・槓で牌を晒す、鳴き間違い、多牌 
* 軽罰符（1000点供託）: 誤ポン・槓発声のみ、リーチ取消、少牌 

## 【フェーズ1：タスク2】 差分比較用データ定義（Mリーグ / 雀魂 / 天鳳）

* **1. Mリーグ 競技規則（四人麻雀）**
* 持ち点/返し点: 25,000点持ち / 30,000点返し
* ウマ: 10-30
* トビ終了: なし
* 切り上げ満貫: あり
* ダブロン・トリロン: なし（頭ハネ）
* 途中流局: 一切なし

* **2. 雀魂 段位戦ルール（四人麻雀）**
* 持ち点/返し点: 25,000点持ち / 30,000点返し
* ウマ: 5-15
* トビ終了: あり（0点未満）
* 切り上げ満貫: なし
* ダブロン・トリロン: あり（トリロンも全員アガリ）
* 途中流局: あり（九種九牌、四風連打、四開槓、四家立直）

* **3. 天鳳 段位戦ルール（四人麻雀）**
* ウマ: 10-20
* トリロン: 流局（三家和）
* 途中流局: あり（九種九牌、四風連打、四開槓、四家立直、三家和）
